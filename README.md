# PedaGo

**Educational Canvas Platform** built with Tauri v2 + Vue 3.

Local-first design workspace with real-time sync, drag-and-drop blocks, rich text editing, and SCORM packaging for LMS.

Powered by the Architectural Handoff Specification.

## Quick Start
1. `pnpm create tauri-app@latest .`
2. Follow the detailed spec for components, stores, and capabilities.

## Features
- Adaptive responsive workspace (Container Queries)
- Tiptap ProseMirror editor
- Yjs + IndexedDB + WebSocket sync
- Pragmatic Drag & Drop
- SCORM 1.2 Export

Built for educators and instructional designers.
